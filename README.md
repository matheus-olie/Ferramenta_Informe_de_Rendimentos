# Ferramenta_Informe_de_Rendimentos
Desafio realizado no Bootcamp Santander, pela DIO. A atividade requeria a criação de uma ferramenta para auxiliar o investidor com a declaração de IR.
------------------------------------------------------------------------------------------------------------------------
A ferramenta conta com:
* Menu Interativo (Não disponível na versão web do excel)
* Formulário de Dados do Titular
* Tela de informes de Rendimentos
* Tela de Notas e extratos Bancários

OBS: OS DADOS PREENCHIDOS SÃO SIMULADOS, ou seja, são artificiais, apenas para imitar um preenchimento da planilha.
